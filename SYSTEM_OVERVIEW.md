## 🔬 TRIANGULAR RESEARCH EXPLAINED

### The Three-Source Verification

Every critical requirement is validated against:

1. **Canadian Authority** (Primary)
   - CSA, NBC, OBC
   - "What must we do?"
   - Most stringent in Canadian jurisdiction

2. **International Best Practice** (Validation)
   - AASHTO, ASTM, Eurocode
   - "What do others do?"
   - Industry consensus approaches

3. **Provincial/Municipal Compliance** (Regulatory)
   - OBC, MTO, OPS
   - "What does jurisdiction require?"
   - Local regulatory requirements

### Decision Rule

**When requirements differ:**
```
IF CSA requirement > AASHTO requirement
   → ADOPT CSA (Canadian jurisdiction priority)
   → Document AASHTO as secondary check
   → Note: "CSA more stringent, AASHTO review recommended for edge cases"

ELSE IF AASHTO requirement > CSA requirement
   → ADOPT AASHTO for critical applications
   → Document rationale in design report
   → Note: "Both codes acceptable, AASHTO used for conservatism"

ELSE IF Provincial/Municipal requirement differs
   → ADOPT Local requirement (highest)
   → Document override in specification
   → Note: "Local authority requirement supersedes standard practice"
```

### Example Application

**CONCRETE STRENGTH SPECIFICATION**
```
Requirement: Bridge deck concrete strength

Source 1 - CSA A23.3-14
  → f'c ≥ 35 MPa minimum
  → Testing per CSA A23.1-14 Section 7.2

Source 2 - AASHTO LRFD 5.4.2.4
  → f'c ≥ 35 MPa (same)
  → Testing per ASTM C39

Source 3 - OBC 2020
  → Defers to National Building Code
  → NBC defers to CSA standards

DECISION:
  ✓ SPECIFY: f'c = 35 MPa minimum
  ✓ TESTING: Per CSA A23.1-14, Section 7.2
  ✓ VERIFICATION: ASTM C39 as secondary check
  ✓ CITATION: "CSA A23.3-14 Section 8.2.1 and CSA A23.1-14 Section 7.2"
```
